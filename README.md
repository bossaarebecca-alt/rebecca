# Portfolio Futuriste - Rebecca Bossa

## Instructions pour personnaliser votre portfolio

### 1. Ajouter votre photo de profil
- Créez une image nommée `placeholder-profile.jpg` (ou modifiez le nom dans index.html)
- Dimensions recommandées: 400x400 pixels (carré)
- Format: JPG ou PNG
- Placez le fichier à la racine du projet

### 2. Ajouter votre CV
- Créez un fichier PDF nommé `cv-rebecca-bossa.pdf`
- Placez-le à la racine du projet
- Le bouton "Télécharger mon CV" permettra aux visiteurs de le télécharger

### 3. Personnaliser les informations
Modifiez les sections suivantes dans `index.html`:
- Vos compétences dans la section "Compétences"
- Vos projets dans la section "Projets" 
- Vos liens sociaux (GitHub, LinkedIn, Twitter)

### 4. Caractéristiques du design futuriste
- **Couleurs néon**: Cyan (#00d4ff), Violet (#7b2cbf), Rose (#ff006e)
- **Effets de lueur**: Ombres portées colorées et brillantes
- **Animations**: Rotation, pulsation, scintillement
- **Typographie**: Orbitron (titres) et Rajdhani (texte)
- **Arrière-plans**: Dégradés avec effets de profondeur
- **Cartes interactives**: Effets de survol avec transformations

### 5. Fonctionnalités incluses
- Photo de profil avec effet de lueur rotative
- Bouton de téléchargement de CV
- Coordonnées complètes (email, téléphone, localisation)
- Formulaire de contact fonctionnel (simulation)
- Navigation responsive avec menu mobile
- Animations au défilement
- Design entièrement responsive

## Structure des fichiers
```
/workspace
├── index.html          # Page principale

├── script.js           # JavaScript pour les interactions
├── placeholder-profile.jpg  # À ajouter (votre photo)
└── cv-rebecca-bossa.pdf     # À ajouter (votre CV)
```
